# Assistant Bot - Address Book & Notes Manager

Чат-бот помічник з командного рядка для управління контактами та нотатками. Цей Python додаток надає інтерактивний інтерфейс для зберігання, пошуку та управління контактною інформацією разом з особистими нотатками.

## Огляд

Це чат-бот з командного рядка на базі Python, який допомагає користувачам керувати своєю адресною книгою та особистими нотатками. Бот має модульну архітектуру з маршрутизацією команд, постійним зберіганням даних та комплексними можливостями управління контактами.

## Встановлення

1. Клонуйте або завантажте цей репозиторій
2. Переконайтеся, що у вас встановлено Python 3.7+
3. Зовнішні залежності не потрібні (використовується лише стандартна бібліотека Python)

## Використання

### Запуск бота

Запустіть головний скрипт:

```bash
python main.py
```

Ви побачите привітальне повідомлення:
```
Welcome to the assistant bot! Type 'help' to see commands.
>

## Структура проекту

```
bez-tabu-project/
├── main.py              # Точка входу та головний цикл
├── router.py            # Система маршрутизації та диспетчеризації команд
├── handler.py           # Обробники команд для контактів та нотаток
├── address_book.py      # Клас AddressBook з управлінням контактами та нотатками
├── contact.py           # Клас Contact з логікою валідації
├── note.py              # Клас Note з генерацією унікальних ID
├── storage.py           # Шар серіалізації та збереження даних
├── cli.py               # Утиліти CLI (незавершено)
└── data/
    └── address_book.pkl # Файл постійного зберігання
```
