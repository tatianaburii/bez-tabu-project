# Персональний помічник (Assistant Bot - Address Book & Notes Manager)

Персональний помічник з командним рядком для управління контактами та нотатками. Цей Python додаток надає інтерактивний інтерфейс для зберігання, пошуку та управління контактною інформацією разом з особистими нотатками.

## Огляд

Це персональний помічник з командним рядком на базі Python, який допомагає користувачам керувати своєю адресною книгою та особистими нотатками. Бот має модульну архітектуру з маршрутизацією команд, постійним зберіганням даних та комплексними можливостями управління контактами.

## Встановлення

1. Клонуйте або завантажте цей репозиторій
2. Переконайтеся, що у вас встановлено Python 3.7+
3. Зовнішні залежності не потрібні (використовується лише стандартна бібліотека Python)

## Використання

### Запуск бота

Запустіть головний скрипт:

```bash
python main.py
```
або встановіть пакет 

pip install -e .

та введіть в командному рядку

bez-tabu



Ви побачите привітальне повідомлення:
```
Welcome to the assistant bot! Type 'help' to see commands.
>
```
